How To Orient Yourself

Campus center: Frank Sinatra / Library
North side: Humanities
South side: Social Sciences
East side: Law + Rothberg
Main arrival edge: Main Gate
  • If you feel lost, first ask someone for Frank Sinatra Square or the library.
  • When moving between faculties, returning to the central pedestrian area is usually the easiest choice.
  • After you reach the correct faculty zone, use the local block signs for the exact classroom.

Best First Stop

If this is your first time on campus, the safest navigation strategy is: reach the central library area, confirm your orientation, then continue to your faculty.

The Bloomfield Library is officially described as being between the Faculties of Humanities and Social Sciences, which makes it a very useful mental midpoint.

This page is designed as a practical first-arrival guide, not a formal architectural map. It keeps the wording simple so new students can move confidently between the main landmarks. For an official reference, use the Hebrew University campus maps page.
